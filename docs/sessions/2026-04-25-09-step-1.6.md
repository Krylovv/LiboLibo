## Что сделано

### Брендовый цвет

[`Theme/Theme.swift`](../../LiboLibo/Theme/Theme.swift) — единственный токен:
```swift
static let red = Color(red: 1.0, green: 61.0/255.0, blue: 61.0/255.0)  // #FF3D3D с libolibo.me
```
+ extensions `Color.liboRed` и `ShapeStyle.liboRed` (для `.tint(.liboRed)`, `.foregroundStyle(.liboRed)`).

В [`App/LiboLiboApp.swift`](../../LiboLibo/App/LiboLiboApp.swift) — `.tint(.liboRed)` на корне. Этот единственный модификатор красит выбранный таб, NavigationLink-стрелки, кнопку «Подписаться», play/pause в mini-player, прогресс-слайдер и пилюли в плеере. Apple HIG: один `.tint` на root + системные шрифты везде.

### Шрифт

Сначала натянул Gerbera-trial — но trial-веса оказались с битым Й (без бреве в кириллице), Илья сказал «не нравится, давай бесплатный по гайдам». Откатил всё на системный San Francisco через стандартные text styles SwiftUI (`.subheadline`, `.title3`, `.caption`, и т.д.) — Dynamic Type работает из коробки.

### Подкасты — лентой, не сеткой

[`PodcastsView.swift`](../../LiboLibo/Features/Podcasts/PodcastsView.swift) переписан с `LazyVGrid` на `List`. Каждая ячейка — обложка 64×64 + название + издатель. Это и привычнее (Apple Podcasts тоже список), и решает претензию из 1.4 про «съезжающую левую колонку» окончательно.

### Описания из RSS

- [`Models/Podcast.swift`](../../LiboLibo/Models/Podcast.swift) — добавил `PodcastChannelInfo { description }`.
- [`Services/RSSParser.swift`](../../LiboLibo/Services/RSSParser.swift) теперь возвращает `(channel, episodes)`. Парсит channel-level `<description>` и `<itunes:summary>` отдельно от item-уровневых. HTML-сущности и теги стрипаются прямо в парсере.
- [`Services/PodcastsRepository.swift`](../../LiboLibo/Services/PodcastsRepository.swift) — `fetchFeed(for:)` отдаёт оба, `loadAllEpisodes()` берёт только эпизоды.
- [`PodcastDetailView.swift`](../../LiboLibo/Features/Podcasts/PodcastDetailView.swift) рендерит **полное** описание подкаста под кнопкой «Подписаться».

### Превью в Фиде и в детали подкаста

Расширение `String.firstSentences(maxCount:)` в RSSParser.swift:
- Берёт первое предложение.
- Если в нём меньше 2 слов — добавляет второе (правило Ильи).

[`EpisodeRow`](../../LiboLibo/Features/Feed/EpisodeRow.swift) теперь по умолчанию показывает превью между заголовком выпуска и датой. Параметр `showsPreview: Bool = true` — на случай, если где-то нужно компактнее.

### Поиск

[`SearchView.swift`](../../LiboLibo/Features/Search/SearchView.swift):
- Матчит по подкаст-названию, **артисту** (`libo/libo`) — добавлено по запросу.
- Эпизоды — по title, summary, podcastName.
- В `.task` догружает выпуски, если пользователь зашёл сразу в Поиск, минуя Фид.
- Две секции: Подкасты + Выпуски.

### Плеер

[`PlayerView.swift`](../../LiboLibo/Features/Player/PlayerView.swift):
- **Blurred-artwork backdrop** — обложка эпизода рендерится за всем контентом с `blur(radius: 80, opaque: true)` + затемняющий слой `Color.black.opacity(0.25)`. Текст всегда белый (`.foregroundStyle(.white)`) — читается на любой обложке, белой/чёрной/цветной.
- `.preferredColorScheme(.dark)` на корне sheet'а — статус-бар и системные элементы тоже в тёмной палитре.
- **Скорость** и **таймер сна** — теперь pill-кнопки (44pt min height), переключаются циклически по тапу:
  - Скорость: 1× → 1.25× → 1.5× → 2× → 0.8× → 1×.
  - Сон: Сон → 15м → 30м → Сон.
- Активные пилюли — на `Color.liboRed.opacity(0.85)`, неактивные — `Color.white.opacity(0.18)`.
- Большая play/pause — белая иконка 72pt в центре.

[`PlayerService`](../../LiboLibo/Services/PlayerService.swift):
- `enum SleepTimer: { off, fifteen, thirty }` + `cycleSleepTimer()` + `setSleepTimer(_:)`.
- Таймер реализован как `Task` со `Task.sleep(N минут)` → автопауза при срабатывании.
- `cycleSpeed()` — вращает по `Self.speedOptions`.

### Mini-player

[`MiniPlayerView.swift`](../../LiboLibo/Features/Player/MiniPlayerView.swift):
- Кнопки play/pause и skip-back — 44pt hit zone (HIG требование), визуал — иконки внутри.
- Play/pause иконка — `.foregroundStyle(.liboRed)`.
- Без собственного фона — `tabViewBottomAccessory` даёт liquid glass.
